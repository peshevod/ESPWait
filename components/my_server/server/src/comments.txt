1. Install
  1.1. Install Node.js
  1.2. npm install vue -g
       npm install ajax -g
  1.3. npm uninstall vue-cli -g
       npm install -g @vue/cli
  1.4. save public and src directories
       delete server directory
       cd my_server
       vue create server
       cd server
       vue add vuetify
  1.4. restore public and src directories
  1.5. npm install --save axios vue-router vuex
 
2. To test and develop
	- set server in store.js to 'http://192.168.1.25/'

        - in my_server.c  uncomment:
 		function start_my_server: esp_err_t ret = httpd_start(&my_server, &config);
		function stop_my_server: httpd_stop(my_server);

	- in my_server.c  comment:
 		function start_my_server: esp_err_t ret = httpd_ssl_start(&my_server, &config);
		function stop_my_server: httpd_ssl_stop(my_server);

	- npm run serve
	- http://localhost:8080

3. Production
	- set server in store.js to ''

	- in my_server.c  uncomment:
 		function start_my_server: esp_err_t ret = httpd_ssl_start(&my_server, &config);
		function stop_my_server: httpd_ssl_stop(my_server);

        - in my_server.c  comment:
 		function start_my_server: esp_err_t ret = httpd_start(&my_server, &config);
		function stop_my_server: httpd_stop(my_server);


	- npm run build
	- copy content of dist to directory server on sdcard
